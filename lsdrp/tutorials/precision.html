Precision Tool and SmartSnap Guide

In today's guide/tutorial, I will show you how to use the Precision Tool and SmartSnap! This will be a very simple tutorial. Although you may think it's just "right click and prop go brrrrr", it goes deeper than that. I'll explain every detail (or at least try to) without making it too complicated. Let's start!

Rotate

With the "Rotate" Tool Mode on, it's self-explanatory: it rotates props. This can be useful if you're trying to rotate a prop like models/props_building_details/Storefront_Template001a_Bars.mdl so you can see through it. Using the physics gun wouldn't really cut it, so all you need to do is click on the prop with left click, and rotate it. It's similar to using the physgun while holding down E, but a bit more precise!

Move

The "Move" Tool Mode is also very straightforward but incredibly useful! For example, let's say you want to place a door prop, but the gap isn't perfect. The Move Tool Mode comes in handy here! To use it, click on any prop (like a barrel) to create a ghost-form of the prop. Then click again to position or rotate it as needed. This is useful for builders and helps with precise placements that can be tricky with the physgun.

Some Tool Modes, like "Weld" or "Removal," are less useful. For example, the "Removal" Tool Mode can often be replaced by the "Move" Tool Mode.

SmartSnap

SmartSnap is similar to the Precision Tool, but instead of rotating or moving, it creates brackets that your crosshair snaps to when you hold the interact key (default is E). Though SmartSnap has issues, like the brackets not showing, here's how to fix it.

Open the spawn menu (default Q), click on Options, and scroll to the bottom under the "Player" category to find SmartSnap. Once you're there, ensure that "Use click grid" is enabled. If not, enable it. If it's enabled and the brackets are missing, disable it, and your brackets should appear. You can customize SmartSnap further to have as few or as many brackets as you'd like.

SmartSnap is just as useful as the Precision Tool, though it’s sometimes underrated because of the bracket issue. Now you know how to fix it!
